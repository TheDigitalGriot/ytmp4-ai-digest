How should the comparison results be organized?

Pick the structure that fits how you'd naturally explore a multi-video analysis

A

Topic-Centric

Organized by themes/topics that Claude extracts across all videos. Each section shows what each video said about that topic, with timestamps and screenshots. Great for "what does everyone think about X?"

Example structure:
• Unified Summary
• Topic: "Scaling Laws" → Video A [04:21], Video C [12:03]
• Topic: "Safety Concerns" → Video A [08:15], Video B [03:40]
• Where They Disagree
B

Video-Centric

Each video gets its own section with summary, key moments, and screenshots. A separate "Cross-Analysis" tab shows common themes and differences. Good for exploring each video individually first.

Example structure:
• Overview Dashboard
• Tab: Video A → summary, timeline, screenshots
• Tab: Video B → summary, timeline, screenshots
• Tab: Cross-Analysis → common topics, differences
C

Dashboard with Both Views

A top-level dashboard with unified summary and topic map, then toggle between topic-centric and video-centric views. Most flexible but more complex UI.

Example structure:
• Dashboard: summary + topic map + key stats
• View toggle: [By Topic] / [By Video]
• Either view shows timestamps + screenshots
• Click any moment → grab additional screenshot
